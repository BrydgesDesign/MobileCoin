The MobileCoin Foundation coordinates and encourages a global community of developers working together to co-create the simplest possible private payments network.

Protocol

The MobileCoin Protocol is a set of rules that define a decentralized cryptocurrency. Anyone is free to review the open soure code that implements the protocol, to verify that it is correct or to propose improvements. The MobileCoin Foundation will oversee development of the MobileCoin software in GitHub.
